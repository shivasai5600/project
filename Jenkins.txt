---How to automate testcases from GitHub in Jenkins --------
1. Create a job in Jenkins
2. In Source Code Management >> select 'git'
   >> git repo url
   >> credentials by clicking add button
3. select branch '%main'
4. in BUILD >> select Execute Window batch window
5. in command -- pytest --browser %browser% --url %url% --html=reportgit.html
   >> for this we need to Parametrize >> choose parametrize
   5.1 >> browser
      chrome
      edge
      firefox
   5.2 >> url
      http://yatra.com
      http://ga.yatra.com
6. Run build now
